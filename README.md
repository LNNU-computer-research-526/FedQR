# Learning to Aggregate in Federated Learning via Local Beneficial Knowledge Consistency



Our code is based on Python version 3.11 and PyTorch version 2.9.1. You can run the code with the following command:

```
python main.py --dataset cifar10 --local_model ResNet20 --server_method fedqr --client_method local_train --server_epochs 5
```
